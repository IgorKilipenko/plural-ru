# plural-ru


Склонение русских существительных и глаголов!

### Установка

```
npm install --save plural-ru
```


Примеры:

```js
const {
    noun,
    verb
} = require('plural-ru');

// склонение существительного
noun(1, 'файл', 'файла', 'файлов'); // файл
noun(2, 'файл', 'файла', 'файлов'); // файла
noun(5, 'файл', 'файла', 'файлов'); // файлов

// склонение существительного + плейсхолдер
noun(1, '%d файл', '%d файла', '%d файлов'); // 1 файл
noun(2, '%d файл', '%d файла', '%d файлов'); // 2 файла
noun(5, '%d файл', '%d файла', '%d файлов'); // 5 файлов

// единственное и множественное число (если переданы 2 склонения)
noun(1, 'Оформить товар', 'Оформить товары'); // Оформить товар
noun(21, 'Оформить товар', 'Оформить товары'); // Оформить товары

// склонение глаголов
noun(1, 'нашлась', 'нашлись', 'нашлось'); // нашлась
noun(2, 'нашлась', 'нашлись', 'нашлось'); // нашлись
noun(5, 'нашлась', 'нашлись', 'нашлось'); // нашлось
```

Сложный пример из жизни

```js
const found = noun(
    foundCount,
    'нашлась',
    'нашлись',
    'нашлось'
);
const ideas = verb(
    foundCount,
```
